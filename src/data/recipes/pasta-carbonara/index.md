- { ingredient: "Pecorino romano / Parmigiano reggiano", comments: "Al gust" }
- { quantity: "3", ingredient: "Clares d'ou", comments: "2/3 ous per persona"}
- { ingredient: "Perbre" }
- { ingredient: "Sal" }
steps:
  - Posa en una olla aigua a bullir i quan ho faci afegeix sal i cou la pasta. Un cop cuita la pasta reserva l’aigua de la coccio.
  - Talla a cuadrats petits el guanciale i posal a daurar en una paella a foc mig. Un cop cuit reserva el guanciale i deixa el greix a la paella.
  - Separa el rovells de les clares i reservals en un bol. Afegeix als rovells el pecorino ratllat, un toc de sal, pebre molt i remena-ho fins que sigui una pasta homogenia.
  - Aboca la pasta a la paella on havies cuinat el guanciale, aboca una mica d’aigua de coccio i remena per lligar el greix.
  - Incorpora la mescla dels rovells a la paella i remena fins a obtenir una salsa homogenia, si cal afegeix mes aigua.
