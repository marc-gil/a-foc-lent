difficulty: Facil

servings: 4
ingredients: 
  - { quantity: "1L", ingredient: "Brou de pollastre", comments: "Idealment dasi, pero es pot substituir per brou" }
  - { quantity: "4-6", ingredient: "Shiitake deshidratat" }
  - { quantity: "3", ingredient: "Ceba dolça" }
  - { quantity: "1", ingredient: "Poma Fuji", comments: "O qualsevol altra varietat, preferiblement amb un toc àcid" }
  - { quantity: "3 cda", ingredient: "Ketchup" }
  - { quantity: "3 porcions", ingredient: "Pastilla de curry japonés" }
  - { quantity: "25ml", ingredient: "Salsa perrins" }
  - { quantity: "15ml", ingredient: "Salsa de soja fosca", comments: "Substituible per soja normal i una mica de sucre" }
  - { quantity: "4", ingredient: "Pots d'arros" }
  - { quantity: "4", ingredient: "Lloms de porc", comments: "Substituible per xulles" }
  - { quantity: "2", ingredient: "Ou", }
  - { ingredient: "Farina" }
  - { ingredient: "Panko" }
  - { ingredient: "Mantega", comments: "opcional" }
  - { ingredient: "Oli" }
  - { ingredient: "Sal" }
  - { ingredient: "Pebre" }
steps:
  - Salpebrar el llom per ambdues cares i deixar reposar a la nevera 30-45 mins destapat.
  - Escalfar el brou de pollastre fins que comenci a bullir. Afegir els shiitakes, apagar el foc i tapar per a que infusioni durant uns 20 min.
  - Tallar la ceba a juliana i caramel·litzar juntament amb mantega, oli i sal durant 45 mins aproximadament a foc mig fluix (fins que tingui un color marro fosc). Anar remenant i afegint afegint aigua a mesura que es comenci a enganxar a la paella.
  - Un cop la ceba esta caramelitzada, pelar la poma, ratllar-la i afegir-la juntament amb el ketchup i amb les pastilles de curry. Remoure durant dos minuts a foc mig fluix.
  - Retirar el shiitake del brou i integrar-lo poc a poc amb la mescla de la ceba i el curry. Un cop estigui tot integrat afegir la salsa perrins i la salsa de soja i tiruruar-ho tot fins que quedi una salsa fina.
  - Posar oli a escalfar a 180º, mentrestrant agafar el llom, passar-lo per farina, ou i panko i fegir-lo.
  - Escalfar els pots d'arros 1 min al microones i emplatar juntament amb el curry i el llom.
